## **📥 Installation**  
1. Clone the repo:  
   ```sh
   git clone https://github.com/vanshsharma69/testimonials.git
   ```
2. Install dependencies:  
   ```sh
   npm install
   ```
3. Run locally:  
   ```sh
   npm start
   ```
